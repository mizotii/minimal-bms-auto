【コメント】-----------------------------------------

こんにちは、Kyuzo Sameuraです。珍しく超！〆切ギリギリ！
ハウス党で立候補したのはいいものの、3曲ぐらいボツになって
うわ〜〜どうしようどうしようとなって何とかひねり出しました。
そういえば合作でブラジリアンな曲は何度か出しましたが、
個人で出すのは何気に初めてかも。

タイトルの「ceu」は本来eの文字に点がつきます。
ポルトガル語で空ですね。セーウと読みます。
セーウ、それは君が見た光。

時間が足りないのでボーカルを頼むに頼めず、NEUTRINOに頼りました。
結構自然な感じで歌ってくれて凄いですね。
（しかし、なぜ時間が無いのにボーカル曲を…？）

歌詞書くの基本的にヘタクソなんですが、
いつもモチーフが楽園に行きてえ…みたいな感じなんですよね。
深層心理が垣間見える見える、（闇が）深いぜ。

キー音が余りに余ってるのですが、
曲調的に同梱譜面は優し目にしております。
というより強い譜面の作り方わからん…。

毎度の如く、差分、動画の公開などはご自由にどうぞ。
それではよろしくお願いします〜！！
